class PyFlyt.gym_envs.quadx_envs.quadx_hover_env.QuadXHoverEnv(sparse_reward: bool = False, flight_dome_size: float = 3.0, max_duration_seconds: float = 10.0, angle_representation: str = 'quaternion', agent_hz: int = 40, render_mode: None | str = None, render_resolution: tuple[int, int] = (480, 480))#

Simple Hover Environment.

Actions are vp, vq, vr, T, ie: angular rates and thrust

The target is to not crash for the longest time possible

Reward:

-100 for collisions or out of bounds, -0.1 otherwise

class PyFlyt.gym_envs.quadx_envs.quadx_waypoints_env.QuadXWaypointsEnv(sparse_reward: bool = False, num_targets: int = 4, use_yaw_targets: bool = False, goal_reach_distance: float = 0.2, goal_reach_angle: float = 0.1, flight_dome_size: float = 5.0, max_duration_seconds: float = 10.0, angle_representation: str = 'quaternion', agent_hz: int = 30, render_mode: None | str = None, render_resolution: tuple[int, int] = (480, 480))#

QuadX Waypoints Environment.

Actions are vp, vq, vr, T, ie: angular rates and thrust

The target is a set of [x, y, z, yaw] targets in space

Reward:

200.0 for reaching target, -100 for collisions or out of bounds, -0.1 otherwise

compute_state()#

Computes the state of the current timestep.

This returns the observation as well as the distances to target. - “attitude” (Box) —– ang_vel (vector of 3 values) —– ang_pos (vector of 3/4 values) —– lin_vel (vector of 3 values) —– lin_pos (vector of 3 values) —– previous_action (vector of 4 values) —– auxiliary information (vector of 4 values) - “target_deltas” (Sequence) —– list of body_frame distances to target (vector of 3/4 values)
